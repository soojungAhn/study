##### find
    - name : 파일 이름으로 검색한다.
    - type : 파일 종류로 검색한다.
    - user : 지정한 소유자가 소유한 모든 파일을 검색한다.
    - prem : 지정한 권한과 일치하는 파일을 검색한다.
    - size : 지정한 크기의 파일을 검색한다.

##### grep
    -n : 찾고자 하는 문자열의 행까지 출력한다.
    -v : 해당하는 문자열을 제외하고 출력한다.
    -A : 찾는 문자열 + 다음 n행
    -B : 찾는 문자열 + 이전 n행
    -r : 하위 디렉토리를 포함한 모든 파일에서 검색한다.
